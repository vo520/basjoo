# Basjoo 项目概述

## 1. 项目简介

**Basjoo** 是一个开源的 AI 客服系统，支持多租户、多 AI 提供商、知识库管理和可嵌入 Widget。

### 核心特性
- 🤖 **多 AI 提供商**: 支持 OpenAI、DeepSeek、Gemini、Anthropic 等
- 📚 **知识库管理**: URL 爬取、文件上传、Qdrant 向量检索
- 💬 **流式对话**: SSE 实时响应，支持多轮对话
- 🔒 **安全可靠**: SSRF 防护、CORS 控制、API 密钥加密
- 🌐 **嵌入组件**: 可嵌入网站的客服 Widget
- 🐳 **Docker 部署**: 一键部署到 Linux 服务器

## 2. 技术栈

### 后端
| 技术 | 说明 |
|------|------|
| FastAPI | Python Web 框架 |
| SQLAlchemy (async) | 异步 ORM |
| SQLite | 轻量级数据库 |
| Redis | 缓存和限流 |
| Qdrant | 向量数据库 |
| Scrapling | 网页爬取微服务 |

### 前端
| 技术 | 说明 |
|------|------|
| Next.js 14 | React 框架 |
| React 18 | UI 库 |
| TypeScript | 类型安全 |
| i18next | 国际化 |

### Widget
| 技术 | 说明 |
|------|------|
| TypeScript | 类型安全 |
| esbuild | 打包工具 |
| SSE | 服务器推送事件 |

## 4. 主要功能模块

### 4.1 AI 对话模块
- 流式 SSE 响应
- 多轮对话上下文
- 来源引用标注
- 人工接管机制

### 4.2 知识库模块
- **URL 管理**: SSRF 安全验证、自动爬取调度
- **文件上传**: PDF、TXT、CSV、DOCX 等
- **向量检索**: Qdrant 相似度搜索
- **多租户隔离**: 租户级别的数据隔离

### 4.3 认证授权模块
- JWT Token 认证
- 角色权限控制 (super_admin, admin, support)
- API 密钥加密存储

### 4.4 调度任务模块
- URL 自动抓取 (每小时检查)
- 历史记录清理 (凌晨 3 点)
- 会话自动关闭 (30 分钟无活动)
- Agent 软删除清理 (24 小时周期)

## 7. 安全特性

1. **SSRF 防护**: 阻止 localhost、内网 IP、RFC1918 地址
2. **Widget 源白名单**: 公共 API 强制域名验证
3. **CORS 控制**: 区分 `Origin: null` 和缺失 Origin
4. **API 密钥加密**: Fernet 对称加密
5. **任务锁机制**: 防止索引和抓取冲突
6. **限流保护**: Redis 支持的请求限流

## 8. 部署方式

### 一键部署 (Ubuntu/Debian)
```bash
curl -fsSL https://raw.githubusercontent.com/haoyiyin/basjoo/main/install-deploy.sh | sudo sh
```

### Docker Compose
```bash
# 开发环境
docker compose --profile dev up -d

# 生产环境
docker compose --profile prod up -d
```

### 本地开发
```bash
# 后端
cd backend && python3 main.py

# 前端
cd frontend-nextjs && npm run dev

# Widget
cd widget && npm run dev
```
