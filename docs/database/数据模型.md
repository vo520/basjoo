## 2. 模型定义

### 2.1 Workspace (工作空间)

工作空间是顶级组织单元，包含多个 Agent 和管理员。

```python
class Workspace(Base):
    __tablename__ = "workspaces"
    
    id: int                    # 主键
    name: str                  # 工作空间名称
    owner_email: str           # 所有者邮箱 (唯一)
    created_at: datetime       # 创建时间
```

**字段说明**:
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK | 主键 |
| name | String(100) | NOT NULL | 工作空间名称，默认 "Default Workspace" |
| owner_email | String(255) | UNIQUE, NOT NULL | 所有者邮箱 |
| created_at | DateTime | Server Default | 创建时间 |

**关系**:
- `agents`: 一对多 Agent
- `quotas`: 一对一 WorkspaceQuota
- `admin_users`: 一对多 AdminUser

### 2.2 Agent (智能体)

核心实体，代表一个 AI 客服机器人。

```python
class Agent(Base):
```

**字段说明**:

| 字段 | 类型 | 默认值 | 说明 |

| workspace_id | Integer | - | 工作空间 ID |
| name | String(100) | "AI Agent" | 名称 |
| description | Text | NULL | 描述 |
| agent_type | String(50) | "website_support" | 类型 |
| channel_mode | String(50) | "web_widget" | 渠道模式 |
| avatar | String(500) | NULL | 头像 URL |
| system_prompt | Text | "You are a helpful..." | 系统提示词 |
| model | String(100) | "gpt-4o-mini" | 模型名称 |

| api_key | String(500) | NULL | API 密钥 (加密) |

| provider_type | Enum | "openai" | 提供商类型 |
| embedding_provider | String(20) | "jina" | Embedding 提供商 |

**关系**:
- `workspace`: 多对一 Workspace

### 2.3 URLSource (URL 知识源)

存储被抓取的网页信息。

```python
class URLSource(Base):
    __tablename__ = "url_sources"
    
    id: int                    # 主键
    agent_id: str              # 外键，关联 Agent
    url: str                   # 原始 URL
    normalized_url: str        # 规范化 URL (去重)
    status: Enum               # 状态
    title: str                 # 页面标题
    content: Text              # 清洗后的正文
    content_hash: str          # 内容哈希 (去重)
    is_indexed: Boolean        # 是否已索引
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| agent_id | String(50) | Agent ID |
| url | String(1000) | 原始 URL |
| normalized_url | String(1000) | 规范化 URL (小写、去www) |
| status | Enum | pending/fetching/success/failed |
| last_fetch_at | DateTime | 最后抓取时间 |
| last_error | Text | 最后错误信息 |
| title | String(500) | 页面标题 |
| content | Text | 清洗后的正文 |
| content_hash | String(64) | SHA256 哈希 |
| fetch_metadata | JSON | 抓取元数据 (ETag, Last-Modified) |
| is_indexed | Boolean | 是否已训练索引 |

**status 枚举**:
- `pending`: 待抓取
- `fetching`: 抓取中
- `success`: 抓取成功
- `failed`: 抓取失败

**索引**:
- `ix_url_sources_agent_status`: (agent_id, status)
- `uq_agent_normalized_url`: UNIQUE (agent_id, normalized_url)

### 2.4 KnowledgeFile (知识文件)

存储上传的知识文件。

```python
class KnowledgeFile(Base):
```

**status 枚举**:

- `processing`: 处理中

- `failed`: 失败
- `pending`: 待处理

### 2.5 ChatSession (聊天会话)

```python
class ChatSession(Base):
    __tablename__ = "chat_sessions"
    
    id: str                    # 主键，格式 sess_<12位hex>
    agent_id: str              # Agent ID
    session_id: str             # 客户端会话 ID
    status: str                # 状态
    visitor_id: str            # 访客 ID
    visitor_ip: str            # 访客 IP
    visitor_country: str       # 访客国家
    message_count: int         # 消息数
    total_tokens: int          # 总 Token 数
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String(50) | 主键 |
| agent_id | String(50) | Agent ID |
| session_id | String(100) | 客户端会话 ID |
| locale | String(10) | 语言 |
| visitor_id | String(100) | 访客标识 |
| visitor_ip | String(50) | 访客 IP |
| visitor_country | String(50) | 访客国家 |
| visitor_region | String(50) | 访客省份 |
| visitor_city | String(50) | 访客城市 |
| status | String(20) | active/taken_over/closed |
| message_count | Integer | 消息数 |
| total_tokens | Integer | 总 Token 数 |

**status 枚举**:
- `active`: 活跃
- `taken_over`: 已被人工接管
- `closed`: 已关闭

**索引**:
- `uq_chat_sessions_active_session`: UNIQUE (agent_id, session_id) WHERE status != 'closed'
- `ix_chat_sessions_agent_session`: (agent_id, session_id)

## 3. 索引

### 3.1 唯一约束

| 表 | 约束名 | 字段 |
|------|--------|------|
| workspaces | (owner_email) | UNIQUE |
| agents | (id) | PRIMARY KEY |
| url_sources | uq_agent_normalized_url | (agent_id, normalized_url) |
| chat_sessions | uq_chat_sessions_active_session | (agent_id, session_id) WHERE status != 'closed' |
| agent_members | uq_agent_member_admin | (agent_id, admin_user_id) |
| admin_users | (email) | UNIQUE |
| tenants | (slug) | UNIQUE |
| knowledge_bases | (qdrant_collection) | UNIQUE |

### 3.2 普通索引

| 表 | 索引名 | 字段 |

| agents | idx_agents_workspace | workspace_id |
